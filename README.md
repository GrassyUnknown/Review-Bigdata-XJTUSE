# Review-Bigdata 点评大数据分析及应用系统

## 项目介绍
### 简介
本项目能够对点评大数据进行处理，功能包括：
- 分析点评数据并查看分析结果
- 查看商户列表和商户详情、分类筛选商户
- 查看商户的简介、评分、评论
- 查看用户的简介、评分、评论
- 以ALS搜索推荐算法搜索商户
- 查看好友列表和推荐好友
- 查看个人信息
- 登录与登出

### 项目目录
#### /Vue
前端代码，基于vue技术实现，使用npm运行，详见README
#### /Springboot
后端代码，基于Java和Springboot实现，需要连接数据库运行
#### /database
数据库相关文件，可根据其中提示导入数据
#### /ALS_model
ALS搜索推荐算法相关文件和代码
#### /Jira
项目管理过程中的一些文件
#### /Zeppelin代码及展示
利用Zeeplin处理大数据的结果pdf
